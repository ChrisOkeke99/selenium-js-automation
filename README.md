# Selenium JS Automation (Herokuapp)

A small UI automation project using **Selenium WebDriver with JavaScript**, written to demonstrate core automation testing concepts.

## What this project covers
- Automated login tests for https://the-internet.herokuapp.com/login
- Valid and invalid credential scenarios
- Explicit waits and assertions
- Headless Chrome execution
- Screenshot capture on test failure

## Tech stack
- Node.js
- selenium-webdriver
- Mocha
- Chai

## Setup
```bash
npm install
```

## Run tests
```bash
npm test
```
